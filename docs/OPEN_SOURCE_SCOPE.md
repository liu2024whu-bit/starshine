# Open-source scope

Starshine is the public, independently maintained core of a broader spatial-intelligence research effort.

## Included

- original source code selected for stable public contracts;
- synthetic GeoJSON created for this repository;
- automated tests;
- public architecture, security, contribution, and roadmap documents.

## Excluded

- unreleased research modules;
- private PostgreSQL/PostGIS databases and credentials;
- full application UI and internal deployment configuration;
- large experimental rasters and course-delivery artifacts;
- textbook PDFs, OCR output, and materials without explicit redistribution permission;
- historical local logs, personal paths, and generated runtime output.

The exclusion boundary is not intended to create a temporary showcase. It allows this repository to remain permanently public while the maintainers continue research and release reviewed components over time.
