# Roadmap

## 0.1 — Public core

- [x] GeoJSON validation
- [x] CRS-aware buffering
- [x] dissolve and point-within-polygon summary
- [x] bounded JSON workflow execution
- [x] command-line demo and synthetic data
- [x] automated tests and CI

## 0.2 — Reproducibility and interoperability

- [ ] GeoPackage input/output adapter
- [ ] stable JSON Schema for workflow files
- [ ] structured diagnostics with step identifiers
- [ ] result provenance manifest
- [ ] benchmark fixtures for small vector workflows

## 0.3 — Community research preview

- [ ] additional overlay and proximity operators
- [ ] optional PostGIS adapter with read-only discovery
- [ ] plugin boundary for independently validated operators
- [ ] documentation examples for education and research replication

The public repository will remain intentionally smaller than the unreleased research system. Features move here only after licensing, data, security, and reproducibility review.
